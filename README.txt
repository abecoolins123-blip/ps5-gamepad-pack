PS5 DualSense Gamepad Resource Pack for Eaglercraft U_35

Usage:
1. Place the ZIP in your resourcepacks folder.
2. Make sure your mobile API is injected first (for touch and alternate input support).
3. Connect your PS5 DualSense controller via USB or Bluetooth.
4. Launch Eaglercraft U_35. The controller should now work:
   - Left stick → movement (WASD)
   - Right stick → camera
   - Cross → Jump
   - Circle → Sneak
   - Square → Inventory
   - Triangle → Drop
   - L1/R1 → Sprint / Attack
   - Create / Options → Menu / Enter
